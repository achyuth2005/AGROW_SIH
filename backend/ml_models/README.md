# ML Models Directory

This directory contains **Python-based ML/AI models and pipelines** that are **NOT** part of the Flutter app.

## Contents

| Folder | Description |
|--------|-------------|
| `Heatmap_Service/` | Heatmap generation with CNN + LLM analysis |
| `SAR Model/` | Sentinel-1 SAR crop health prediction |
| `S2_Pipeline/` | Sentinel-2 vegetation indices pipeline |
| `Sentinel 2_pipeline/` | Alternative Sentinel-2 analysis pipeline |
| `TIme_series/` | AutoNHITS time series forecasting |
| `Updated_Chatbot/` | LLM-powered agricultural chatbot |

## Note

These models are deployed on **Hugging Face Spaces** and accessed via HTTP API from the Flutter app.

See [INTEGRATION_GUIDE.md](./INTEGRATION_GUIDE.md) for API endpoints and usage.
